    g-font-layout

    @mixin g-font-layout($fontSizePx, $lineHeightType: 0, $layout: 'L25', $addPx: 4) { ... }
    View source

    Description

    N00のレイアウトに対応したフォントサイズを返すg-fontの拡張mixin
    レイアウト文字列に'N00'が入ったときには、基準フォントサイズにaddPx（デフォルト4px）を加算したフォント指定を生成する。
    'N00'以外では基準フォントサイズのままを返す。

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $fontSizePx

    基準となるフォントサイズ

    		Number— none
    $lineHeightType

    lineheightタイプ

    		Number0
    $layout

    レイアウト文字列

    		String'L25'
    $addPx

    N00のときに加算するpx値

    		Number4

    Example

    // 標準利用ケース
.t0-b-hogeHoge {
  &__bd {
    @include g-font-layout(28, 1, 'N00');
  }
}
// g-font(32, 1)の結果と同じ
    // 加算する数字を変えた場合
.t0-b-hogeHoge {
  &__bd {
    @include g-font-layout(28, 1, 'N00', 6); // 加算する数字を4px → 6pxに変更
  }
}
// g-font(34, 1)の結果と同じ

    Output

    レイアウトに対応したg-font実行結果を返す

    Author

    • NADAMOTO(@wms-nad)

    g-textOuterGlow

    @mixin g-textOuterGlow($size: 1, $color: #fff, $blur-radius: '1px') { ... }
    View source

    Description

    photoshopでいうところの「光彩（外側）」を文字に追加するmixin。
    引数指定がない場合は、サイズ1,#fff,ぼかしの半径1pxの光彩になる。

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    光彩のサイズ。1と2から選択する

    		Number1
    $color

    光彩の色

    		Color#fff
    $blur-radius

    光彩のぼかしの半径

    		String'1px'

    Example

    // 標準利用ケース
.t0-b-sample {
  p {
    @include g-textOuterGlow;
  }
}
// g-font(1, #fff, 1px)の結果と同じ
    // サイズ、色、ぼかしの半径を指定する場合
.t0-b-sample {
  p {
    @include g-textOuterGlow(2, #000, 3px);
  }
}

    Output

    指定したサイズ、色、ぼかしのtext-shadowのプロパティ値を返す

    Author

    • 鈴木慎太郎(@ShintarouSuzuki)
