    g-font-layout

    @mixin g-font-layout($fontSizePx, $lineHeightType: 0, $layout: 'L25', $addPx: 4) { ... }
    View source

    Description

    N00のレイアウトに対応したフォントサイズを返すg-fontの拡張mixin
    レイアウト文字列に'N00'が入ったときには、基準フォントサイズにaddPx（デフォルト4px）を加算したフォント指定を生成する。
    'N00'以外では基準フォントサイズのままを返す。

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $fontSizePx

    基準となるフォントサイズ

    		Number— none
    $lineHeightType

    lineheightタイプ

    		Number0
    $layout

    レイアウト文字列

    		String'L25'
    $addPx

    N00のときに加算するpx値

    		Number4

    Example

    // 標準利用ケース
.t0-b-hogeHoge {
  &__bd {
    @include g-font-layout(28, 1, 'N00');
  }
}
// g-font(32, 1)の結果と同じ
    // 加算する数字を変えた場合
.t0-b-hogeHoge {
  &__bd {
    @include g-font-layout(28, 1, 'N00', 6); // 加算する数字を4px → 6pxに変更
  }
}
// g-font(34, 1)の結果と同じ

    Output

    レイアウトに対応したg-font実行結果を返す
